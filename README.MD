# AI驱动的自动化测试平台

这是一个智能化的Web应用自动化测试平台，支持多AI引擎集成，能够自动生成测试用例并执行自动化测试。

## 功能特性

- AI驱动的智能测试用例生成
- 支持多AI引擎（DeepSeek、Claude、OpenAI等）
- 自动化登录与页面导航
- 智能元素识别与数据填充
- 自动化脚本生成（Python Selenium）
- 完整的测试报告生成

## 安装与使用

1. 克隆项目
2. 安装依赖：`npm install`
3. 复制环境文件：`cp .env.example .env`
4. 配置AI API密钥
5. 启动服务：`npm start`

## 配置

请在 `.env` 文件中配置所需的AI服务API密钥。
